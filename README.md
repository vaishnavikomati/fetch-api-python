# Fetch API Data Using Requests

## Objective
Fetch API data using Python requests module and display results properly.

## Features
- Uses requests module
- Parses JSON response
- Search/filter functionality
- Displays output clearly

## Technologies Used
- Python
- Requests
- JSON

## API Used
https://jsonplaceholder.typicode.com/users

## How to Run
pip install requests
python api_fetch.py
